# Crew

> AI-powered task management for everyone - Engineers, PMs, and Business teams working together with AI agents.

## Quick Start

Run Crew instantly without installation:

```bash
npx crew
```

This will launch the application locally and open it in your browser automatically.

## What is Crew?

Crew is a modern task management tool that brings AI coding agents to everyone on your team. It helps you organize projects with kanban-style task management while providing powerful integrations with AI coding agents.
